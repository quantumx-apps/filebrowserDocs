Settings under `frontend` control branding, layout toggles, styling, and sidebar links. They correspond to the `frontend` section in `config.generated.yaml` in the main repository.

## Topics

- {{< doclink path="configuration/frontend/interface/" text="Interface" />}} — sidebar links, storage bars, nav buttons
- {{< doclink path="configuration/frontend/branding/" text="Branding" />}} — name, description, favicon, login assets, external links
- {{< doclink path="configuration/frontend/styling/" text="Styling" />}} — global CSS, backgrounds, event themes
- {{< doclink path="configuration/frontend/themes/" text="Themes" />}} — per-user selectable themes

**Related:** Default **user** UI preferences (locale, preview toggles, permissions template for new users) live under `userDefaults`, documented in {{< doclink path="configuration/users/" text="User management" />}}, not under `frontend`.
